My first step into direct shader programming in Unity. This project was inspired by Jasper Flick's [CatLike Coding](https://catlikecoding.com/unity/tutorials/flow/) Unity series, and Garrett Gunnell's [Water Rendering](https://github.com/GarrettGunnell/Water) project. While Unity has shader graph that you can drag and drop items into, I found it more engaging to actually program each part. This project really helped me understand the Unity render pipeline.
